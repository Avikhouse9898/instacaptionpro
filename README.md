# InstaCaptionPro

InstaCaptionPro is a tiny demo for generating Instagram captions with OpenAI.

## Local development

1. Clone and install dependencies:
   ```bash
   git clone <repo-url>
   cd instacaptionpro
   npm install
   ```
2. Set your OpenAI API key:
   ```bash
   export OPENAI_API_KEY=your_key_here
   ```
3. Start a local server to serve the app (any static server or Next.js dev server works).

## Tests

Run the test suite with:

```bash
npm test
```
